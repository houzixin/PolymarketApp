## ⚠️ 待解决

### Kotlin 版本冲突

**问题**: React Native Gradle Plugin 需要 Kotlin 1.9.0，但项目配置为 1.8.0

**解决方案**:

**方案 A**: 升级 Kotlin 到 1.9.0
```bash
cd /root/.openclaw/workspace/PolymarketApp/android
# 修改 build.gradle
ext.kotlinVersion = "1.9.0"
classpath("org.jetbrains.kotlin:kotlin-gradle-plugin:1.9.0")

# 重新编译
./gradlew clean assembleDebug
```

**方案 B**: 降级 React Native 到 0.72.x
```bash
cd /root/.openclaw/workspace/PolymarketApp
npm install react-native@0.72.0 --save
cd android
./gradlew clean assembleDebug
```

**方案 C**: 使用预编译 APK (推荐)
- 在已配置好的开发环境中编译
- 或使用 GitHub Actions CI/CD
