## 🔧 编译 APK 的方法

### 方法 1: 本地编译 (最快)

**要求**: 已配置 Java + Android SDK 的电脑

```bash
# 1. 解压项目
tar -xzf PolymarketApp-Full.tar.gz
cd PolymarketApp

# 2. 安装依赖
npm install --force

# 3. 编译
cd android
./gradlew assembleDebug

# 4. APK 位置
# android/app/build/outputs/apk/debug/app-debug.apk
```

### 方法 2: GitHub Actions (推荐)

```bash
# 1. 推送到 GitHub
cd PolymarketApp
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/你的用户名/PolymarketApp.git
git push -u origin main

# 2. 等待自动编译 (15-20 分钟)
# 访问：https://github.com/你的用户名/PolymarketApp/actions

# 3. 下载 APK
```

### 方法 3: 找人代编译

**把项目包发给有 Android 开发环境的人**:
```bash
# 发送文件
scp PolymarketApp-Full.tar.gz user@dev-pc:~/

# 对方编译
ssh user@dev-pc "cd ~ && tar -xzf PolymarketApp-Full.tar.gz && cd PolymarketApp/android && ./gradlew assembleDebug"

# 下载 APK
scp user@dev-pc:~/PolymarketApp/android/app/build/outputs/apk/debug/app-debug.apk .
```
